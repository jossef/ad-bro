### How to install?

- Download the extension - [link](https://github.com/jossef/annoying-ynet/releases/download/1.0.0/annoying-ynet.crx)
- Open Chrome Settings
- Click Extensions
- Drag & Drop the file youv'e just downloaded and that's it
